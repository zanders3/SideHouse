Lightmap Packer

The Lightmap Packer is an Editor Extension for Unity Game Engine that displays the position and size of Lightmap UVs for all static objects in a Unity scene. It has recently been released on the Unity Asset Store.

It allows you to optimise your lightmap UV usage by adjusting the lightmap scales of each object in your scene quickly, with all changes made displayed instantly.

Users can expect to get 85-90% lightmap texel usage up from the 20-30% found by default. The tool can also help debug light bleeding issues by showing exactly where a specific mesh is located in the lightmap.
